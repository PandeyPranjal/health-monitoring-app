# HealthPulse

HealthPulse is a mobile-first full-stack health monitoring web application that allows users to track health metrics, simulate wearable device integration, and manage doctor appointments.

## Features

- User authentication (JWT-based)
- Onboarding flow
- Health tracking (heart rate, sleep, steps, SpO2)
- Demo data & simulated device connection
- Appointment booking system
- Mobile-first UI

## Tech Stack

- Frontend: React (Vite)
- Backend: Django + DRF
- Database: SQLite (dev)
- Auth: JWT

## Deployment

- Frontend: Vercel
- Backend: Render

## How it works

Users sign up → complete onboarding → connect device or use demo → view health dashboard → book appointments.

## Limitations

- No real wearable integration (simulated)
- No real-time sync

## Future Scope

- Fitbit / Google Fit integration
- AI health anomaly detection
- Real-time notifications
